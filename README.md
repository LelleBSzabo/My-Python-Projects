# 🐍 Python Programming Fundamentals Project

This project was completed during a **Data Technician Bootcamp** and focuses on learning the core fundamentals of **Python programming**. The objective was to build foundational programming skills by writing scripts that perform calculations, process logic, and interact with users.

The project demonstrates how Python can be used to create simple programs using **variables, conditional logic, loops, and user input**, helping develop the problem-solving skills required for data and software development tasks.

---

## 🚀 Project Overview

In this project, Python was used to:

- Write scripts that perform **basic calculations**
- Create **interactive programs** that accept user input
- Build **logical conditions** to control program flow
- Automate repetitive tasks using loops
- Solve classic programming challenges such as **FizzBuzz**

The exercises focused on developing **logical thinking and programming structure**, which are essential skills for data-related roles.

---

## 🛠️ Python Skills Demonstrated

### Variables

Variables were used to store and manipulate data within Python programs.

Examples include:

- Storing numbers for calculations
- Saving user input values
- Tracking loop counters and conditions

Example:

```python
age = 25
name = "Alex"
```

---

### User Interaction

Programs were designed to interact with users using:

- `print()` to display information
- `input()` to collect user input

Example:

```python
name = input("Enter your name: ")
print("Hello", name)
```

This allows programs to become **interactive rather than static scripts**.

---

### Type Casting

Type casting was used to convert data types when performing calculations or processing user input.

Examples include converting text input into numbers:

```python
age = int(input("Enter your age: "))
```

This ensures data can be used correctly within calculations and program logic.

---

### Conditional Logic

`if`, `elif`, and `else` statements were used to control program behaviour based on different conditions.

Example:

```python
if score >= 50:
    print("Pass")
else:
    print("Fail")
```

Conditional logic allows programs to **make decisions based on data**.

---

### Loops

Loops were used to automate repetitive tasks and iterate through sequences.

#### For Loops

Used to repeat a block of code a set number of times.

```python
for i in range(5):
    print(i)
```

#### While Loops

Used when repetition continues until a condition changes.

```python
n = 1
while n <= 10:
    print(n)
    n += 1
```

---

## 🔢 Programming Challenge: FizzBuzz

One of the exercises involved implementing the classic **FizzBuzz programming challenge**, which demonstrates the use of loops and conditional logic.

Example solution:

```python
n = 1

while n <= 100:
    if n % 3 == 0 and n % 5 == 0:
        print("fizzbuzz")
    elif n % 3 == 0:
        print("fizz")
    elif n % 5 == 0:
        print("buzz")
    else:
        print(n)

    n += 1
```

This exercise demonstrates:

- Loop control
- Conditional logic
- Modulo operations
- Structured problem solving
